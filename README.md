# ReadComicOnline rcostation.xyz temporary patch

Unofficial debug-signed patch of the ReadComicOnline Mihon/Tachiyomi extension.

Only install this if you are comfortable using an unofficial debug-signed extension APK.

Changed the extension base URL from:

```text
https://readcomiconline.li
```

to:

```text
https://rcostation.xyz
```

## Install

Uninstall the existing ReadComicOnline extension first, then install this APK.

Mihon may mark it as untrusted. Trust it manually after installation.

## Notes

This is not an official Mihon, Tachiyomi, or Keiyoushi build. It is a temporary patch until the upstream extension is updated.
